## 과제 필수 화면 요소

커뮤니티 메인 화면에 아래 요소가 모두 포함되어야 합니다.

- 게시글 피드
- 게시글 카드
- 작성자 정보
- 게시글 유형 필터
- 반응 또는 댓글 진입
- 글쓰기 진입

---

## 배경 및 목적

SEDA의 현재 학습 경험은 개인 중심이다. 학습 동료와 연결되고 진도를 공유하는 커뮤니티 공간을 통해 **사용자 락인을 강화하고 서비스 확장의 발판**을 마련한다.

---

## 게시글 유형

커뮤니티에서 작성할 수 있는 게시글은 3가지 유형이다.

| 유형 | 설명 | 예시 |
|---|---|---|
| 학습 질문 | EPS-TOPIK 관련 학습·시험 질문 | "이 문장에서 조사가 왜 '에서'인가요?" |
| 한국 생활 | 한국 생활 팁, 문화, 일상 공유 | "외국인 등록증 발급 방법 정리했어요" |
| 취업 정보 | EPS-TOPIK 합격 후 취업·비자 관련 정보 | "H-2 비자 갱신 경험 공유합니다" |

피드 상단에 유형 필터가 칩(Chip) 형태로 노출된다: **전체 · 학습 질문 · 한국 생활 · 취업 정보**

---

## 사용자 스토리 (MVP 범위)

- 학습자로서, 모르는 내용을 질문 게시글로 올려 다른 학습자의 도움을 받고 싶다
- 학습자로서, 한국 생활 팁을 공유하거나 다른 나라 학습자의 경험을 읽고 싶다
- 학습자로서, 취업·비자 정보를 다른 학습자와 나누고 싶다
- 학습자로서, 원하는 유형의 게시글만 모아 보고 싶다
- 학습자로서, 좋은 게시글에 좋아요를 눌러 반응을 남기고 싶다
- 학습자로서, 게시글에 댓글로 피드백을 주고받고 싶다
- 학습자로서, 다른 언어로 작성된 게시글을 내 언어로 번역해 읽고 싶다
- 학습자로서, 새 글을 작성하고 싶다

## 핵심 기능

### 피드 (커뮤니티 메인 화면)

- 하단 탭바에 커뮤니티 탭 신설 — 마이 탭 바로 왼쪽 (탭 아이콘은 후보자 자유)
- 전체 학습자의 게시글 목록 (최신순 정렬, 무한 스크롤)
- 상단 유형 필터 칩 (전체 · 학습 질문 · 한국 생활 · 취업 정보)
- 플로팅 버튼(FAB)으로 글쓰기 진입

---

## 게시글 카드 구성

### 작성자 정보

| 항목 | 설명 |
|---|---|
| 프로필 이미지 | 없을 경우 이니셜 아바타로 대체 |
| 닉네임 | 필수 표시 |
| 게시글 유형 배지 | 학습 질문 / 한국 생활 / 취업 정보 |
| 작성 시간 | 24시간 이내: "2시간 전" / 이후: "5월 21일" |

### 게시글 콘텐츠

피드 카드에서 표시하는 내용은 아래와 같다.

| 항목 | 규칙 |
|---|---|
| 이미지 | 첨부 시 표시 (선택, 1장) |
| 게시글 제목 | 필수 |
| 본문 미리보기 | 최대 2줄, 초과 시 "···" 처리 |
| 전체 본문 | 최대 500자 (글쓰기 기준) |

### 반응 및 댓글

| 항목 | 규칙 |
|---|---|
| 반응 종류 | 좋아요(♥) 1종 (MVP), 토글 가능 |
| 반응 수 / 댓글 수 | 카드 하단 표시 |

### 다국어 번역

피드에는 11개 언어로 작성된 게시글이 혼재된다.

- 사용자의 앱 설정 언어와 **다른 언어**로 작성된 카드에 **"번역 보기"** 텍스트 버튼 노출
- 탭 시 카드 내 본문이 번역 텍스트로 인라인 교체 (별도 페이지 이동 없이)
- 번역 상태에서는 **"원문 보기"** 버튼으로 전환 가능

## 이 과제 범위 밖 (디자인하지 않아도 됨)

- 게시글 상세 화면
- 글쓰기 화면
- 알림 시스템
- 팔로우/팔로워
- 신고/차단
- 게시글 수정/삭제
- 검색/해시태그
- 프로필 상세 화면

## 성공 지표 (참고용)

- 커뮤니티 탭 도입 후 사용자 평균 세션 체류 시간 증가
- 전체 활성 사용자 중 커뮤니티 탭 진입률
- 커뮤니티 진입 사용자의 주간 재방문율

---

## 기술 제약 (참고용)

- iOS + Android 네이티브 (SwiftUI / Kotlin)
- UI 텍스트는 11개 언어로 번역 처리됨
- 게시글 본문: 원문 노출 + 인라인 번역 기능 제공
- 번역 시 텍스트 길이가 달라지므로 카드 높이는 고정하지 않음
- 사용자 보유 정보: 닉네임, 프로필 이미지, 국적

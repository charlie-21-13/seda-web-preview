# SEDA 앱 개요

## 서비스 정의

SEDA는 EPS-TOPIK(고용허가제 한국어능력시험) 학습 모바일 앱이다.

- **대상**: EPS-TOPIK 응시를 준비하는 16개 송출국 외국인 노동자 (성인)
- **플랫폼**: iOS + Android 네이티브 앱
- **지원 언어**: 11개 (English, Nepali, Vietnamese, Bahasa Indonesia, Thai, Khmer, Bengali, Tagalog, Urdu, Russian, 한국어)

## 현재 탭 구조

| 탭 | 주요 내용 |
|---|---|
| 홈 | 오늘 학습 요약, 추천 콘텐츠 |
| 학습 | 유형별 연습, 모의 연습 |
| 활동 | 오답노트, 학습 히스토리 |
| 마이 | 설정, 계정 |

## 스크린샷 사용 안내

`03-app-screenshots/` 폴더의 스크린샷(홈, 학습, 마이페이지)은 앱의 기능 구조와 정보 밀도를 이해하기 위한 참고 자료입니다.

- 기존 시각 스타일을 그대로 복제할 필요는 **없습니다**.
- 다만 SEDA가 가진 학습 앱으로서의 신뢰감, 다국적 사용자(16개국·11개 언어) 맥락, 모바일 네이티브 구현 가능성은 고려해주세요.
- 커뮤니티 화면에 적합한 최소 디자인 언어는 후보자가 직접 제안해주세요.

## 최소 브랜드 키워드

커뮤니티 화면의 시각 방향을 정의할 때 아래 키워드를 기준으로 삼아주세요.

- **신뢰감 있는** — 학습 결과·성장을 책임진다는 인상
- **학습에 집중되는** — 시각 노이즈를 줄이고 학습 흐름을 방해하지 않는
- **성인 학습자 톤** — 진지한 목적을 가진 성인 사용자에게 어울리는 (게임스럽거나 유아적이지 않은)
- **다국적 사용자에게 중립적인** — 특정 문화권에 치우치지 않는
- **다시 오고 싶은** — 매일 짧게라도 돌아오게 되는

## 주의사항

- 지나치게 게임스럽거나 아동용처럼 보이지 않게 설계해주세요.
- 학습 앱이지만 EPS-TOPIK 시험 대비라는 진지한 목적을 가진 사용자임을 고려해주세요. 커뮤니티가 학습 동기를 해치지 않도록 균형을 잡아주세요.
- 11개 언어 번역 시 텍스트가 길어지거나 줄바꿈 위치가 달라질 수 있음을 고려해주세요.

# SEDA 커뮤니티 탭 프로토타입

Threads형 피드 구조 + SEDA 디자인 토큰을 적용한 HTML 프로토타입입니다.

## 실행 방법

`index.html`을 브라우저에서 열면 됩니다. 별도 빌드 없음.

```bash
open index.html
```

## Typography

| 구분 | 영문 | 크기 |
|------|------|------|
| Body | SF Pro Text | 19px 이하 |
| Display | Poppins | 20px 이상 (SEDA 로고 등) |
| 한글 | Pretendard | fallback |

## 포함 범위 (PRD)
- 하단 GNB **커뮤니티** 탭 (마이 왼쪽)
- 게시글 피드 · 카드 · 작성자 정보
- 유형 필터 칩 (전체 / 학습 질문 / 한국 생활 / 취업 정보)
- 좋아요 · 댓글 진입 · FAB 글쓰기
- 다국어 **번역 보기** / **원문 보기** (인라인)

## 미포함 (과제 범위 외)
- 홈 · 학습 · 활동 · 마이 실제 화면
- 게시글 상세 · 글쓰기 전체 화면
